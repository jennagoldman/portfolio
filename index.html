Here I've provided a sample of my work on some of the projects that I'm the most proud. This is an active list that will change over time as I continue to work on cool, new stuff!

PDX Secret Menu

Link to live app coming soon!

PDX Secret Menu allows restaurants to sign up and register their businesses, populate current offerings, and create polls to find out what foodies are interested in seeing from them in the future. Foodies are also able to sign up, vote on polls, and purchase live offerings for scheduled pickup from their favorite restaurants.

Tech Used: React with Redux, Node.js, MongoDB, Express, integrations with Google Maps and Stripe

Social Distance-Ping



Twilio-based SMS service for receiving location-specific COVID-19 web- scraped statistics. Personal contributions include implementing app's core SMS functionality, and pair-programming a new web scraper when initial scraping source unexpectedly shut down.

Tech Used: Node, Express, MongoDB, Puppeteer, Twilio, Heroku

Sew Organized



Full-stack application with user auth that allows users to inventory their supply of embroidery floss, and generate and save color schemes to inspire future projects. I was involved in all aspects of this mob-programmed project, and my favorite contribution was redesigning the entire front-end to better communicate a clean, organized environment more representative of the solution we provided to our user base.

Tech Used: React, Node, Express, PostgreSQL, Heroku

DisasterProof



DisasterProof is a frontend app that helps users prepare for natural disasters through a guided survey that outputs a tailored preparedness plan. I co-led development of the app's core JavaScript functionality, and owned the wire-framing, style guide development and CSS work.

Tech Used: Vanilla JavaScript, localStorage, HTML, CSS

Libeerary



Solo-project to develop a full-stack app with user auth, integrated with the Untappd API, for logging my favorite adult beverages. Currently working on expanding functionality to allow for categorizing beers in addition to favoriting.

Tech Used: React, Node.js, Express.j PostgreSQL, Heroku
