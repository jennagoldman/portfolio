Portfolio

Here I've provided a sample of my work on some of the projects that I'm the most proud of. This is an active list that will change over time as I continue to work on cool, new stuff!

PDX Secret Menu

Tech used: React, Redux, Node, Express, MongoDB

PDX Secret Menu allows restaurants to sign up and register their businesses, populate current offerings, and create polls to find out what foodies are interested in seeing from them in the future. Foodies are also able to sign up, vote on polls, and purchase scheduled offerings for pickup via Stripe integration.

Sew Organized

Tech used: React, Node, Express, PostgreSQL

Full-stack app with user auth that allows users to inventory their supply of embroidery floss, and generate and save color schemes to inspire future projects. I was involved in all aspects of this team project, and my favorite contribution was redesigning the entire front-end to better communicate a clean, organized environment more representative of the solution we provided to our user base.

Social Distance-Ping

Tech used: React, Node, Express, MongoDB, Twilio, Puppeteer, Heroku Scheduler

Twilio-based SMS service for receiving location-specific COVID-19 web- scraped statistics. Personal contributions include implementing app's core SMS functionality, and pair-programming a new web scraper when initial scraping source unexpectedly shut down.

Libeerary

Tech used: React, Node.js, Express.j PostgreSQL, Heroku

Solo-project to develop a full-stack app with user auth, integrated with the Untappd API, for logging my favorite adult beverages. Currently working on expanding functionality to allow for categorizing beers in addition to favoriting.

Disaster Proof

Tech used: Vanilla JavaScript, localStorage, HTML, CSS

DisasterProof is a frontend app that helps users prepare for natural disasters through a guided survey that outputs a tailored preparedness plan. I co-led development of the app's core JavaScript functionality, and owned the wire-framing, style guide development and CSS work.
